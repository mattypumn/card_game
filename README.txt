This is a simple game with a deck of cards.

You can choose a simple deck  with only value 0-60 
  or    a standard deck with has 2 - Ace and 4 Suits: hearts, clubs, diamonds, spades.

There is a list of options available to the user.  

 1. Draw a card to hand
 2. Place a card in hand on top of deck
 3. Shuffle the deck 
 4. Sort the deck
 5. Print Cards that exist in hand    --of course the user sorts their own hand ! 
 6. Quit


 7.   --secret option--  look at deck!    









COMMANDS TO COMPILE AND RUN:   	

$ make 

$ ./play_cards
